# AIを使用した開発

本記事では、開発におけるAIの仕様についてまとめた記事です。

## 使用可能なAI
- Claude (Anthropic)
- ChatGPT (OpenAI Inc.)
- Gemini (Google LLC)
- GitHub Copilot (GitHub, Inc.)
- Copilot (Microsoft Corp.)

## AIの使用禁止事項
以下のAIの使用を禁止します

- `.env`ファイルを含む環境変数などのファイルをAIに提供すること
- 人間のチェックを通さずにマージすること（PRでレビューをもらう）
- その他著作権法や肖像権に違反する物
