# コミットガイド

本記事では本レポジトリにコミットする時のコミットにおいて気をつけることです。
基本的に真冬七瀬Discordサーバーで使用するため、コミットの言語はすべて**日本語**です。

## 例外
- PRをマージする際
デフォルトのままでいいです。

## コミットの内容
```
<contents> (<issue>)
```
コミットは1行

### <contents>
基本的に**日本語**で短文形式

### <issue>
Issueが関連する場合に記載してください。
半角括弧で囲い、`contents`とは半角スペースで囲ってください。

## コミットの例
```
README.MDの修正 (#12345)
```
